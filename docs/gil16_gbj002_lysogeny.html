GIL16 vs GBJ002 contrast

Author

Eduardo Lozano

Published

May 22, 2026

In order to perform the GIL16 vs GBJ002 contrast, we established a log₂ fold-change (LFC) of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = c("Group", "GIL16", "GBJ002"), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, COG_category, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    COG_category,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, 481 upregulated and 562 downregulated genes were obtained. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. This time, we found many genes both upregulated and downregulated. Among the former, two stand out with extremely low adjusted p-values: GBJ002_04202 (flagellar basal body rod protein) and GBJ002_04203 (phage shock protein A). Among the latter, a significant portion of the DE genes on the chromosome correspond to the prophage region. Additionally, many pBtic235 genes are downregulated, indicating that GIL16 also influences the expression of this phage-plasmid, similar to what is observed with GIL01 in the GIL01 vs GBJ002 contrast and in the study by Pavlin et al. (2022).

Code 
volcano_plot(contrast_df, limits = c(-10, 10))

Enrichment analysis

In this section, we present the results of enrichment analyses based on the COG (Clusters of Orthologous Groups), KEGG (Kyoto Encyclopedia of Genes and Genomes), GO (Gene Ontology) and EC (Enzyme Commission) categories for both upregulated and downregulated genes.

COG

According to COG categories, we only found positive enrichment in translation, ribosomal structure and biogenesis (J), nucleotide transport and metabolism (F) and energy production and conversion (C) due to 96, 26 and 34 DE genes, respectively. Although there are more terms related to defense mechanisms (V) than in the GIL01 vs GBJ002 contrast, statistical significance was not reached this time.

Code 
cog_up <- cog_enrichment(up_genes, all_genes)
cog_down <- cog_enrichment(down_genes, all_genes)

KEGG

According to KEGG categories, we found several positive enrichments and one negative. The positive enrichments were mainly related to metabolic pathways, but also to translation. The negative enrichment was associated with DNA replication and with GBJ002_00997 (replicative DNA helicase), GBJ002_01520 (DNA primase), GBJ002_05128 (DNA polymerase-3 subunit alpha), GBJ002_05130 (DNA-directed DNA polymerase), GBJ002_05182 (Single-stranded DNA-binding protein) and GBJ002_05251 (AAA family ATPase) genes.

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, we only found positive enrichment in processes related to translation, gene expression and proteins and macromolecules metabolic processes.

  • Regarding MF, we only observed a high number of enriched functions, mostly related to the binding of RNA-associated molecules. Related processes are also favoured, such as catalytic activity on nucleic acids and ligase activity.

  • Regarding CC, only positive enrichments were found, mainly related to the cytoplasm, ribosomes and protein complexes.

Enrichment summary

This table contains a summary of the enrichments for each category. The subcategory was used for KEGG and the description for GO. Although you can scroll, I recommend exporting using the Excel button and viewing the table in full screen.

Code 
es_up <- enrichment_summary(up_genes, categories = list(cog_up, kegg_up, go_bp_up, go_mf_up, go_cc_up, ec_1_up, ec_2_up, ec_3_up))
es_down <- enrichment_summary(down_genes, categories = list(cog_down, kegg_down, go_bp_down, go_mf_down, go_cc_down, ec_1_down, ec_2_down, ec_3_down))
es_de <- rbind(es_up, es_down)
rownames(es_de) <- c("UP", "DOWN")
