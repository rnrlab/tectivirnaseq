Time points 60 vs 60-mock contrast

Author

Eduardo Lozano

Published

July 10, 2026

In order to perform the time 60 vs 60-mock contrast, we established a log₂ fold-change (LFC) threshold of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, name = "MMC_TRUE_vs_FALSE", lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, Classification, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    Classification,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, 718 upregulated and 550 downregulated genes were obtained. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. In this contrast, we found a large number of upregulated genes on both the chromosome and pBtic235, but only downregulated genes on the former. Regarding the prophage region, it contains 8 of the 10 most overexpressed genes, the remaining ones being the GBJ002_01858 (glyoxalase/fosfomycin resistance/dioxygenase domain-containing protein) and GBJ002_03410 (diacylglycerol kinase) genes.

Code 
volcano_plot(contrast_df)

Enrichment analysis

In this section, we present the results of enrichment analyses based on the COG (Clusters of Orthologous Groups), KEGG (Kyoto Encyclopedia of Genes and Genomes), GO (Gene Ontology) and EC (Enzyme Commission) categories for both upregulated and downregulated genes.

COG

According to COG categories, we found positive enrichment in replication, recombination and repair (L) and nucleotide transport and metabolism (F) and negative enrichment in cell motility (N). Interestingly, there are numerous downregulated flagellar protein genes, so B. thuringiensis swarming motility could also be affected in the absence of tectivirus (Gillis & Mahillon, 2014). This contrast reflects that the impairment of cell motility is not due to bacterial growth, but is affected by induction with MMC.

Code 
cog_up <- cog_enrichment(up_genes, all_genes)
cog_down <- cog_enrichment(down_genes, all_genes)

KEGG

According to KEGG categories, we found positive and negative enrichments. In the case of the former, they were related to metabolism and replication and repair processes. In the case of the latter, it resulted in depletion of diverse metabolic pathways: amino acid metabolism, antimicrobial resistance, signal transduction, cell motility and quorum sensing.

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, positive enrichment was mainly related to nucleotide metabolism, DNA damage and stress response, while negative enrichment was found in diverse metabolic processes, but mainly related to amino acids.

  • Regarding MF, we found positive enrichment in processes related to energy and nucleic acid metabolism, while the transmembrane transporter activity of organic acids, carboxylic acids and organic anions, in addition to vitamin binding and other enzymatic activities, were depleted.

  • Regarding CC, positive enrichments were only found in the cytosol and the intracellular protein-containing complex.

Enrichment summary

This table contains a summary of the enrichments for each category. The subcategory was used for KEGG and the description for GO. Although you can scroll, I recommend exporting using the Excel button and viewing the table in full screen.

Code 
es_up <- enrichment_summary(up_genes, categories = list(cog_up, kegg_up, go_bp_up, go_mf_up, go_cc_up, ec_1_up, ec_2_up, ec_3_up))
es_down <- enrichment_summary(down_genes, categories = list(cog_down, kegg_down, go_bp_down, go_mf_down, go_cc_down, ec_1_down, ec_2_down, ec_3_down))
es_de <- rbind(es_up, es_down)
rownames(es_de) <- c("UP", "DOWN")
