Times 30 vs 0 contrast

Author

Eduardo Lozano

Published

May 22, 2026

In order to perform the times 30 vs 0 contrast, we established a log₂ fold-change (LFC) threshold of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = c("Time", 30, 0), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, COG_category, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    COG_category,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, 396 upregulated and 361 downregulated genes were obtained. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. This time, there are a large number of DE chromosomal genes, both upregulated and downregulated, as well as upregulated pBtic235 genes. In other words, pBtic235 induction begins to be observed at 30 minutes. Regarding the prophage region, it has upregulated genes among the 10 with the highest positive LFC: GBJ002_00999 (hypothetical protein), GBJ002_01003 (RNA polymerase subunit sigma) and GBJ002_01004 (sporulation sigma factor-processing peptidase).

Code 
volcano_plot(contrast_df, limits = c(-6, 6))

Enrichment analysis

In this section, we present the results of enrichment analyses based on the COG (Clusters of Orthologous Groups), KEGG (Kyoto Encyclopedia of Genes and Genomes), GO (Gene Ontology) and EC (Enzyme Commission) categories for both upregulated and downregulated genes.

COG

According to COG categories, we found positive enrichment in energy production and conversion (C), replication, recombination and repair (L) and coenzyme transport and metabolism (H) due to 28, 21 and 29 DE genes, respectively. This represents 16 more genes in the L category than in the case of the 10 vs 0 minute contrast.

Code 
cog_up <- cog_enrichment(up_genes, all_genes)
cog_down <- cog_enrichment(down_genes, all_genes)

KEGG

According to KEGG categories, we found positive and negative enrichments. In the case of the former, they were mainly related to metabolism, but also to replication and repair processes, specifically nucleotide excision repair, due to GBJ002_01735 (excinuclease ABC subunit C), GBJ002_02363 (excinuclease ABC subunit UvrA), GBJ002_02364 (excinuclease ABC subunit B), GBJ002_03098 (DNA helicase PcrA), GBJ002_03099 (NAD-dependent DNA ligase LigA) and GBJ002_03820 (RNA polymerase recycling motor HelD) genes. In the case of the latter, it resulted in depletion of the sulfur relay system and arginine and proline metabolism pathways.

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, we observed positive enrichment in processes related stress and stimuli response.

  • Regarding MF, we found a positive enrichment in ATP-dependent activity acting on DNA, while the transmembrane transporter activity of organic acids, carboxylic acids and organic anions were depleted.

Enrichment summary

This table contains a summary of the enrichments for each category. The subcategory was used for KEGG and the description for GO. Although you can scroll, I recommend exporting using the Excel button and viewing the table in full screen.

Code 
es_up <- enrichment_summary(up_genes, categories = list(cog_up, kegg_up, go_bp_up, go_mf_up, go_cc_up, ec_1_up, ec_2_up, ec_3_up))
es_down <- enrichment_summary(down_genes, categories = list(cog_down, kegg_down, go_bp_down, go_mf_down, go_cc_down, ec_1_down, ec_2_down, ec_3_down))
es_de <- rbind(es_up, es_down)
rownames(es_de) <- c("UP", "DOWN")
