Strains GIL16 vs GIL01

Author

Eduardo Lozano

Published

July 9, 2026

In order to perform the GIL16 vs GIL01 strains contrast, we established a log₂ fold-change (LFC) of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = c("Strain", "GIL16", "GIL01"), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, Classification, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    Classification,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. Among the upregulated genes, two stand out with extremely low adjusted p-values: GBJ002_04202 (flagellar basal body rod protein) and GBJ002_04203 (phage shock protein A). Among the downregulated genes, several correspond to the chromosomal prophage region, indicating that GIL16 exerts stronger repression than GIL01. Similarly, many pBtic235 genes are downregulated, further suggesting that GIL16 has a stronger repressive effect on phage-related genes than GIL01.

Code 
volcano_plot(contrast_df)

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, we only found positive enrichment related to gene expression, translation and certain metabolic processes.

  • Regarding MF, we only observed positive enrichment in a high number of functions, mostly related to the binding of RNA-associated molecules. Related functions were also favoured, such as catalytic activity on nucleic acids and ligase activity.

  • Regarding CC, only positive enrichments were found, mainly related to the cytoplasm, ribosomes and protein complexes.

EC

The EC classification establishes up to four levels to define enzymes. In this section, the first three were analyzed, although significant results were only found at second level. Specifically, at level 1.2, 8 out of 17 genes were upregulated, which were: GBJ002_00158 (glutamate-5-semialdehyde dehydrogenase), GBJ002_00936 (2-oxoacid:acceptor oxidoreductase subunit alpha), GBJ002_01185 (pyruvate dehydrogenase complex E1 component subunit beta), GBJ002_01186 (pyruvate dehydrogenase E1 component subunit alpha), GBJ002_01379 (3-methyl-2-oxobutanoate dehydrogenase subunit beta), GBJ002_01380 (3-methyl-2-oxobutanoate dehydrogenase subunit alpha), GBJ002_01594 (bifunctional acetaldehyde-CoA/alcohol dehydrogenase) and GBJ002_02336 (type I glyceraldehyde-3-phosphate dehydrogenase).

Tectivirus genes expression

In this section, we modified the tectivirus genes names so that homologous genes could have their expression compared using DESeq2. Besides, we examined their expression using boxplots. The data are presented in CPM (counts per million), which indicates how many counts per million come from a given gene.

Code 
tectivirus <- combine_tectivirus_genes()

tectivirus_counts <- tectivirus$tectivirus_counts
tectivirus_annotation <- tectivirus$tectivirus_annotation

tectivirus_counts[] <- lapply(tectivirus_counts, function(x) if(is.numeric(x)) ifelse(x %% 1 == 0.5, ceiling(x), round(x)) else x)

dds_tectivirus <- DESeqDataSetFromMatrix(countData = tectivirus_counts, colData = metadata, design = ~ Strain)
dds_tectivirus <- DESeq(dds_tectivirus)
dds_tectivirus <- dds_tectivirus[grepl("^(g|dnt)", rownames(dds_tectivirus)), ]

In order to perform the GIL16 vs GIL01 strains contrast, we established a log₂ fold-change (LFC) of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively.

In GIL01, we found that the most highly expressed gene is GIL01_18, which corresponds to the major capsid protein gene.
