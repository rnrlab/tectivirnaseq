Time points 60 vs 0 contrast

Author

Eduardo Lozano

Published

July 9, 2026

In order to perform the time 60 vs 0 contrast, we established a log₂ fold-change (LFC) threshold of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = list(c("Time_60_vs_0", "MMC_TRUE_vs_FALSE")), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, Classification, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    Classification,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, 694 upregulated and 420 downregulated genes were obtained. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. In this contrast, we found a large number of upregulated and downregulated genes on both the chromosome and pBtic235. Regarding the prophage region, it contains 7 of the 10 most overexpressed genes of the chromosome, the remaining ones being the GBJ002_03022 (hypothetical protein), GBJ002_03410 (diacylglycerol kinase) and GBJ002_04475 (glycosyl hydrolase) genes. Regarding GIL16, almost all genes were upregulated and with LFC between 4.90 and 10.19, being GIL16-DNT_01 the only not DE gene. In the plot, if we select only upregulated GIL16 genes, we can distinguish two main zones: GIL16_01 to GIL16_08 genes (and GIL16_35), with LFC < 7.8; and GIL16_10 to GIL16_31 genes (and GIL16_36), with LFC > 9. The first region corresponds mainly to the genes of the P1-P2 tandem promoters, while the second corresponds to those of the P3 promoter. As we can see, genes expressed from the P3 promoter are more upregulated than those expressed from the P1-P2 tandem promoters.

Code 
volcano_plot(contrast_df)

Enrichment analysis

In this section, we present the results of enrichment analyses based on the COG (Clusters of Orthologous Groups), KEGG (Kyoto Encyclopedia of Genes and Genomes), GO (Gene Ontology) and EC (Enzyme Commission) categories for both upregulated and downregulated genes.

According to COG categories, we only found positive enrichment in energy production and conversion (C) and inorganic ion transport and metabolism (P) due to 32 and 33 DE genes, respectively.

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, we observed positive enrichment in processes related to DNA damage and nucleotide and arginine metabolism, among others.

  • Regarding MF, we only found positive enrichment in processes related to transporter and DNA-associated activities.

EC

The EC classification establishes up to four levels to define enzymes. In this section, the first three were analyzed, achieving statistical significance in all of them for positive enrichment. In the first level, translocases (7) and oxidoreductases (1) numbers were enriched by 15 and 25 DE genes, respectively. In the second, 7.1 and 1.6 levels were enriched by 12 and 6 genes, respectively. Lastly, in the third, 7.1.2 and 1.6.5 levels were enriched by 8 out of 8 genes (F0F1 ATP synthase subunits genes) and 6 out of 10 genes (NADH-quinone oxidoreductase subunits genes).

ec_1_up <- ec_enrichment(up_genes, all_genes, 1)

Enrichment summary

This table contains a summary of the enrichments for each category. The subcategory was used for KEGG and the description for GO. Although you can scroll, I recommend exporting using the Excel button and viewing the table in full screen.

Code
