Strains GIL01 vs GBJ002 contrast

Author

Eduardo Lozano

Published

July 9, 2026

To analyze differential expression across strains and time points, we performed all statistical modeling within a single DESeq2 dataset rather than creating separate DESeq2 objects for each time point. Because the experimental design includes strong temporal dynamics associated with phage induction, library composition changes substantially across conditions. Analyzing all samples together allows DESeq2 to estimate size factors and dispersion parameters using the full dataset, thereby borrowing statistical strength across conditions and ensuring a consistent normalization framework. Importantly, replicate effects (treated as a batch factor in the model) are captured globally; splitting the dataset by time would force DESeq2 to re‑estimate normalization and dispersion independently for each subset, disrupting the shared technical structure of the experiment and potentially introducing artificial differences. Using a unified DESeq2 object therefore provides a coherent, statistically robust foundation that preserves biological comparability, controls technical variation and enables accurate strain‑level contrasts at each time point.

In order to perform the GIL01 vs GBJ002 strains contrast at 30 minutes, we established a log₂ fold-change (LFC) of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = list(c("Strain_GIL01_vs_GBJ002", "StrainGIL01.Time30", "StrainGIL01.MMCTRUE")), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, Classification, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    Classification,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, 15 upregulated and 48 downregulated genes were obtained. In addition, 678 genes were discarded due to low counts, likely because they are expressed in other contrasts. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. Among the upregulated genes, those with the highest LFC are the bacterial large (GBJ002_02755) and small (GBJ002_02756) signal recognition particles ncRNA and the de novo transcript GBJ002-DNT_206. Among the downregulated genes, we find genes from both the chromosome and pBtic235.

Code 
volcano_plot(contrast_df)

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, we observed positive entichment of protein folding and maturation due to nucleotide exchange factor GrpE (GBJ002_01540) and trigger factor (GBJ002_01697) genes.

  • Regarding MF, we found positive enrichments of protein binding functions due to the previous genes.

Code

Enrichment summary

Code
