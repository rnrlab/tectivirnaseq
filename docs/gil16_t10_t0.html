Times 10 vs 0 contrast

Author

Eduardo Lozano

Published

July 2, 2026

In order to perform the times 10 vs 0 contrast, we established a log₂ fold-change (LFC) threshold of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = list(c("Time_10_vs_0", "MMC_TRUE_vs_FALSE")), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, COG_category, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    COG_category,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, only 17 upregulated genes were obtained. In addition, 779 genes were discarded due to low counts, probably because they are expressed in latter times or in the absence of GIL16. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. Although relatively few DE genes are observed, the top two genes with higer LFC are the same that in the analog GIL01 contrast: GBJ002_04258 (DUF3961 domain-containing protein) and GBJ002_04259 (N-acetylmuramoyl-L-alanine amidase) genes. Besides, there are 5 upregulated pBtic235 genes: GBJ002_05237 (hypothetical protein), GBJ002_05238 (hypothetical protein), GBJ002_05239 (nucleoside 2-deoxyribosyltransferase), GBJ002_05241 (metallophosphoesterase) and GBJ002_05242 (hypothetical protein). None of the GIL16 genes were DE, although GIL16-DNT_01 is less expressed at 10 minutes, while almost the rest of the genes have LFC > 1 or even 2 in the case of GIL16_02 (SSB).

Code 
volcano_plot(contrast_df, limits = c(-4, 4))

Enrichment analysis

In this section, we present the results of enrichment analyses based on the COG (Clusters of Orthologous Groups), KEGG (Kyoto Encyclopedia of Genes and Genomes), GO (Gene Ontology) and EC (Enzyme Commission) categories for upregulated genes, as there are no downregulated genes.

COG

According to COG categories, we found no enrichment.

Code

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, we observed enrichments in functions related to DNA damage and stress response.

  • Regarding MF, we found enrichments in processes related to hydrolases and nucleic acid metabolism.

Code

EC

The EC classification establishes up to four levels to define enzymes. In this section, the first three were analyzed, although statistical significance was only achieved in the third (3.1.25) due to upregulated GBJ002_02363 (excinuclease ABC subunit UvrA) and GBJ002_02364 (excinuclease ABC subunit B) genes.

Enrichment summary

This table contains a summary of the enrichments for each category. The subcategory was used for KEGG and the description for GO. Although you can scroll, I recommend exporting using the Excel button and viewing the table in full screen.

Code 
es_up <- enrichment_summary(up_genes, categories = list(cog_up, kegg_up, go_bp_up, go_mf_up, go_cc_up, ec_1_up, ec_2_up, ec_3_up))
es_down <- as.data.frame(matrix(NA, nrow = 1, ncol = 9))
colnames(es_down) <- colnames(es_up)
es_down[1, "Size"] <- 0
es_de <- rbind(es_up, es_down)
rownames(es_de) <- c("UP", "DOWN")
