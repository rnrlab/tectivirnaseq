Time points 60 vs 0 contrast

Author

Eduardo Lozano

Published

July 9, 2026

In order to perform the time 60 vs 0 contrast, we established a log₂ fold-change (LFC) threshold of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = list(c("Time_60_vs_0", "MMC_TRUE_vs_FALSE")), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, Classification, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    Classification,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, 729 upregulated and 654 downregulated genes were obtained. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. In this contrast, we found a large number of upregulated genes on both the chromosome and pBtic235, but only downregulated genes on the former. Regarding the prophage region, it contains 9 of the 10 most overexpressed genes, the remaining one being the GBJ002_03410 (diacylglycerol kinase) gene.

Code 
volcano_plot(contrast_df)

Enrichment analysis

In this section, we present the results of enrichment analyses based on the COG (Clusters of Orthologous Groups), KEGG (Kyoto Encyclopedia of Genes and Genomes), GO (Gene Ontology) and EC (Enzyme Commission) categories for both upregulated and downregulated genes.

COG

According to COG categories, we found positive enrichment in chromatin structure and dynamics (B) and energy production and conversion (C) and negative enrichment in cell motility (N) and amino acid transport and metabolism (E). Interestingly, there are numerous downregulated flagellar protein genes, so B. thuringiensis swarming motility could also be affected in the absence of tectivirus (Gillis & Mahillon, 2014).

Code 
cog_up <- cog_enrichment(up_genes, all_genes)
cog_down <- cog_enrichment(down_genes, all_genes)

KEGG

According to KEGG categories, we found positive and negative enrichments. In the case of the former, they were mainly related to metabolism, but also to replication and repair processes, specifically nucleotide excision repair, due to GBJ002_01735 (excinuclease ABC subunit C), GBJ002_02363 (excinuclease ABC subunit UvrA), GBJ002_02364 (excinuclease ABC subunit B), GBJ002_03098 (DNA helicase PcrA), GBJ002_03099 (NAD-dependent DNA ligase LigA) and GBJ002_03820 (RNA polymerase recycling motor HelD) genes. In the case of the latter, it resulted in depletion of more metabolic pathways than in previous contrast. Interestingly, an enrichment appears in Human diseases, specifically in the beta-lactam resistance pathway.

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, positive enrichments were mainly related to nucleotide metabolism, while negative enrichment was found in the biosynthetic processes of amino acids, carboxylic acids, lipids and other small molecules.

  • Regarding MF, only negative enrichment of transmembrane transporter activity of organic acids, carboxylic acids and organic anions was found.

Enrichment summary

This table contains a summary of the enrichments for each category. The subcategory was used for KEGG and the description for GO. Although you can scroll, I recommend exporting using the Excel button and viewing the table in full screen.

Code
