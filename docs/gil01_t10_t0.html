Times 10 vs 0 contrast

Author

Eduardo Lozano

Published

May 22, 2026

In order to perform the times 10 vs 0 contrast, we established a log₂ fold-change (LFC) threshold of ±0.585. This value reflects an increase or decrease in expression of 50%, depending on whether it is positive or negative, respectively. The default value is 0, but when performing the other contrasts we obtained too many differentially expressed (DE) genes. This violates one of DESeq2’s key assumptions, which is that most genes are not differentially expressed.

Code 
contrast <- results(dds, contrast = c("Time", 10, 0), lfcThreshold = log2(1.5))
contrast_df <- as.data.frame(contrast)

contrast_df <- contrast_df %>%
  { x <- .; x$Gene_ID <- rownames(x); x } %>%
  left_join(
    annotation %>%
      filter(Gene_ID %in% rownames(contrast_df)) %>%
      dplyr::select(Gene_ID, Gene_symbol, Description, COG_category, Association, Type, Subtype, Source),
    by = "Gene_ID"
  ) %>%
  rename(
    Mean_normalized_counts = baseMean,
    LFC = log2FoldChange,
    Adjusted_p_value = padj
  ) %>%
  dplyr::select(
    Gene_ID,
    Gene_symbol,
    Description,
    COG_category,
    Association,
    Type,
    Subtype,
    Source,
    Mean_normalized_counts,
    LFC,
    Adjusted_p_value
  )

As can be seen in the summary() results, only 21 upregulated and 3 downregulated genes were obtained. In addition, 2824 genes were discarded due to low counts, probably because they are expressed in latter times or in the absence of GIL01. Below is an interactive table where all DE genes can be viewed individually.

MA plot

A MA plot visualizes the relationship between the mean expression and the LFC for each gene. It helps to identify genes that are differentially expressed across conditions, highlighting those with large fold changes relative to their expression level.

Code 
plotMA(contrast, alpha = 0.05, xlab = "Mean of normalized counts", ylab = "LFC")

DE genes

DE genes are defined as those with an adjusted p-value below 0.05. Among them, upregulated genes are extracted as those with a LFC > 0.58, while downregulated genes are defined as those with an LFC < -0.58.

Code 
up_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange > log2(1.5) & !is.na(padj)))
down_genes <- rownames(subset(contrast, padj < 0.05 & log2FoldChange < -log2(1.5) & !is.na(padj)))

Volcano plot

A volcano plot displays each gene according to its LFC on the x-axis and the statistical significance (−log₁₀ adjusted p-value) on the y-axis. Although relatively few DE genes are observed, we can highlight the four genes with the highest absolute LFC: on the positive side, GBJ002_04258 (DUF3961 domain-containing protein) and GBJ002_04259 (N-acetylmuramoyl-L-alanine amidase) genes; on the negative side, GBJ002_00878 (1-phosphofructokinase) and GBJ002_00879 (HTH deoR-type domain-containing protein) genes. Although there are no DE genes for pBtic235, we found that the DNA polymerase gene for GIL01 (GIL01_05) is slightly overexpressed (LFC = 1.66, padj = 0.03). The remaining genes expressed from the P1-P2 tandem promoters have LFC > 1, although they are not marked as DE because they do not reach statistical significance. None of the pBtic235 genes were DE.

Code 
volcano_plot(contrast_df, limits = c(-4, 4))

Enrichment analysis

In this section, we present the results of enrichment analyses based on the COG (Clusters of Orthologous Groups), KEGG (Kyoto Encyclopedia of Genes and Genomes), GO (Gene Ontology) and EC (Enzyme Commission) categories for both upregulated and downregulated genes.

COG

According to COG categories, we only found positive enrichment in replication, recombination and repair (L) due to the following genes: GBJ002_00759 (DNA topoisomerase IV subunit A), GBJ002_00760 (DNA topoisomerase IV subunit B), GBJ002_00941 (recombinase RecA), GBJ002_01363 (DNA polymerase IV), GBJ002_02363 (excinuclease ABC subunit UvrA), GBJ002_02364 (excinuclease ABC subunit B), GBJ002_03098 (DNA helicase PcrA) and GBJ002_03904 (helicase-exonuclease AddAB subunit AddB).

Code

GO

The Gene Ontology (GO) is organized into three categories: Biological Process (BP), which describes the biological objectives to which gene products contribute; Molecular Function (MF), which refers to the elemental activities of gene products at the molecular level; and Cellular Component (CC), which specifies the locations within the cell where gene products are active.

  • According to BP, we observed positive enrichments in processes related to DNA damage and stress response.

  • Regarding MF, we found positive enrichments in functions related to energy and nucleic acid metabolism.

Code

Enrichment summary

This table contains a summary of the enrichments for each category. The subcategory was used for KEGG and the description for GO. Although you can scroll, I recommend exporting using the Excel button and viewing the table in full screen.

Code 
es_up <- enrichment_summary(up_genes, categories = list(cog_up, kegg_up, go_bp_up, go_mf_up, go_cc_up, ec_1_up, ec_2_up, ec_3_up))
es_down <- enrichment_summary(down_genes, categories = list(cog_down, kegg_down, go_bp_down, go_mf_down, go_cc_down, ec_1_down, ec_2_down, ec_3_down))
es_de <- rbind(es_up, es_down)
rownames(es_de) <- c("UP", "DOWN")
